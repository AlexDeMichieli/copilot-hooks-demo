# Agent Guardrails

Please follow these rules when working in this repository.

## File Editing
- Only modify files inside the `src/` directory
- Do not edit configuration files in the root directory
- Do not create files outside `src/`

## Dependencies
- Do not install new packages or dependencies
- Work only with existing libraries

## Security
- Never hardcode secrets, API keys, or passwords
- Use environment variables for sensitive configuration
